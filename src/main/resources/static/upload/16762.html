고순도 시카 에센스를 듬뿍 머금은
#시카패드

센텔라 아시아티카의 5가지 핵심 성분인 마데카소사이드, 마데카식애씨드, 아시아티코사이드,
아시아틱애씨드, 병풀 추출물이 피부 고민 부위 케어에 도움을 줍니다.

더 깊고, 진한 보습을 선사하는
#진정보습패드

밀크 타입의 시카 에센스가 깊고, 진한 보습을 전달해
외부 자극으로 예민해진 피부를 진정시켜줍니다.

고순도 정제된 유효 성분으로 탁월한 효과를 선사합니다.

가볍게 착! 집중 시카 케어를 위한 초밀착
#거즈 패드

에센스를 그대로 머금은 거즈 패드가 빈틈없이 피부에 밀착해
시카 에센스의 유효 성분을 효과적으로 흡수시켜
집중적인 시카 케어를 선사합니다.

* 시카 에센스 패드는 통기성이 뛰어나고,
밀착력이 높은 거즈면을 사용해 유효 성분을 효과적으로 전달합니다.

* 내장된 집게를 사용해 위생적이고, 편하게 거즈 패드를 사용하실 수 있습니다.

트루케어 고마주 필링 패드

기능 : 스페셜 각질 케어 (주 1-2회 사용/워시 오프 타입)

소개 : 자연 유래 셀룰로오스와 저자극 필링 성분 PHA의 시너지로 묵은 각질을 부드럽게 제거해주는 #때밀이패드 #지우개패드

트루케어 원스텝 클렌징 패드

기능 : 올인원 클렌징 (노 워시 타입)

소개 : 미셀라 클렌징 워터가 메이크업은 물론 모공 속 노폐물까지 말끔하게 지워주는 #10초클렌징 #세안패드

트루케어 PHA 토너 패드

기능 : 데일리 각질 케어, 피부결 정돈

소개 : 저자극 필링 성분 PHA를 함유해 각질을 케어하고, 피부결을 정돈해주는 #토너패드

트루케어 시카 에센스 패드

기능 : 시카 케어, 진정&보습

소개 : 고순도 시카 에센스를 듬뿍 머금은 거즈 패드가 예민해진 피부의 진정을 도와주는 #시카패드 #거즈패드
